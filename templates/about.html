About Me

From Curiosity to Code — Building for a Safer Internet

I'm Pushprajsinh Upendrasinh Rathod, a fifth-semester Diploma student in Information Technology at Gujarat Technological University (GTU). What started as general curiosity about how things work on the internet turned into a deep passion for cybersecurity after I learned how many people — including family members — fall victim to online scams every year in India.

CyberShield was born from a simple question: why isn't there a free, simple tool in Gujarati-speaking India that helps regular people check if a link or email is dangerous before it's too late? I built it as my answer to that question — combining what I'm learning in my diploma with independent study in web security, networking, and cryptography.

I'm currently in my final year of the Diploma programme and actively preparing to move into cybersecurity as a career — starting with certifications like CEH (Certified Ethical Hacker) and eventually specializing in penetration testing or SOC operations.

Behind the Project

How CyberShield Was Built

A GTU diploma student, a problem worth solving, and a lot of late nights.

💡

The Idea

It started when a family member almost lost money to a fake HDFC SMS. I realized most people have no simple tool to check if a link is safe before clicking. That gap became CyberShield.

🐍

Built with Python & Flask

The backend is built with Python and Flask — what I was learning in my 5th semester. Each tool runs as a separate Flask route with its own detection logic.

🎨

Designed from Scratch

No Bootstrap, no templates. Every pixel of CSS was written by hand — dark mode, animations, responsive layout, and the color system are all custom built using CSS variables.

🔐

Security Logic

The URL scanner checks 8 patterns — HTTPS, IP-based URLs, suspicious TLDs, phishing keywords, encoded characters, domain length, hyphens, and known scam patterns — all in pure Python.

🔒

Privacy First Encryption

The AES-256 encryption tool runs entirely in the browser using the Web Crypto API. Zero data is sent to any server — your text never leaves your device.

🇮🇳

Built for India

Every threat, example, case study, and helpline number is specific to India. UPI fraud, Aadhaar theft, fake job scams — these are Indian problems that needed an Indian solution.

Tech Stack
🐍 Python ⚗️ Flask 🌐 HTML5 🎨 CSS3 ⚡ Vanilla JS 🔐 Web Crypto API 📦 Jinja2

Why CyberShield Exists

The Mission Behind the Tools

India added 600 million internet users in a decade — but cybersecurity awareness didn't scale with adoption. CyberShield is my attempt to close that gap, one free tool at a time.

🆓

Always Free

Every tool on CyberShield is and will remain free. Security should not be a privilege only those who can pay for it deserve.

🔒

Privacy First

Tools like encryption run entirely in your browser. Nothing is sent to any server. Your data stays on your device.

🇮🇳

Built for India

The threats, examples, and guidance are tailored for Indian users — UPI fraud, Aadhaar theft, and local scam patterns.

📚

Education First

Tools alone aren't enough. The awareness guide explains the "why" behind every threat so you recognize the next one.
